Class: Vessel

LBSailSim.Vessel

new Vessel(sailEnv, obj3D) → {LBSailSim.Vessel}

Container representing a vessel that floats. Vessels support:
  • Aerofoils, which are LBSailSim.FoilInstance based objects that are driven by the atmosphere.
  • Hydrofoils,which are LBSailSim.FoilInstance based objects that are driven by the water.
  • Propulsors, which are LBSailSim.Propulsor based objects. TODO: Add ballast, which is just a LBPhysics.RigidBody.
