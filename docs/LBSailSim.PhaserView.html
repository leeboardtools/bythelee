    updateForceArrowFromRigidBody(rigidBody, forceArrow) → {undefined}

    Updates a force arrow from a rigid body's resultant. If the rigid body has a 'getForceArrowResultant' member function defined, that will be called to obtain the resultant, otherwise the rigid body's LBPhysics.RigidBody#getResultant method will be called.

    This is automatically called as a result of calling LBPhaser.PhysicsLink#updateDisplayObjects.

    The getForceArrowResultant function has the same function signature as LBPhysics.RigidBody#getResultant excluding the convertToWrench argument (the first argument).
