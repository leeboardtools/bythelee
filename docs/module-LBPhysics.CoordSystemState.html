LBPhysics.CoordSystemState(velTermsopt) → {module:LBPhysics.CoordSystemState}

new CoordSystemState(velTermsopt) → {module:LBPhysics.CoordSystemState}

This is used to track the transforms for converting between a world coordinate system and a local coordinate system. It can also keep track of the previous transforms so it may calculate the world and local velocities of points in either coordinate system.

calcVectorLocalToWorld(localPos, results, prevLocalPosopt) → {module:LBPhysics.CoordSystemState}

Calculates world position, world velocity, and local velocity for a given local point.
Parameters:
Name Type Attributes Description
localPos module:LBGeometry.Vector3 The local coordinates of the point of interest.
results object The object to receive the results. Results are stored in keys, and are only computed if the key exists:
  • worldPos: The world coordinates of localPos.
  • worldVel: The velocity of localPos in world coordinates.
  • localVel: The velocity of localPos in local coordinates. Note that this is the world velocity rotated into local coordinates, it is not the velocity of the position relative to the local coordinate system (if that were the case then this would always be 0 if prevLocalPos were not defined).
    		•
    prevLocalPos module:LBGeometry.Vector3 <optional>
    		 If defined, the previous local position, used for velocity calculation, if not defined then the previous position is presumed to be localPos.
