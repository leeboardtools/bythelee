Class: MultiLoadCoordinator

LBAssets.MultiLoadCoordinator() → {module:LBAssets.MultiLoadCoordinator}

new MultiLoadCoordinator() → {module:LBAssets.MultiLoadCoordinator}

A class that manages the loading of multiple assets concurrently, calling a callback when all the assets have been loaded.

Typical usage is: var coordinator = new LBAssets.MultiLoadCoordinator(); coordinator.setup(function() { console.log("Load Completed"); }, function() { console.log("Load failed"); }); coordinator.beginLoadCalls(); loader.loadJSON('abc', 'abc.json', coordinator.getOnLoadFunction(), null, coordinator.getOnError()); loader.loadJSON('def', 'def.json', coordinator.getOnLoadFunction(), null, coordinator.getOnError()); coordinator.endLoadCalls();
