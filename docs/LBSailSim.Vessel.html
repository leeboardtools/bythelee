Class: Vessel

LBSailSim.Vessel

new Vessel(sailEnv, obj3D) → {LBSailSim.Vessel}

Container representing a vessel that floats. Vessels support:
  • airfoils, which are LBSailSim.FoilInstance based objects that are driven by the atmosphere.
  • hydrofoils,which are LBSailSim.FoilInstance based objects that are driven by the water.
  • Propulsors, which are LBSailSim.Propulsor based objects.
  • Ballasts, which are LBPhysics.RigidBody based objects.

    For boats, the local coordinate system is presumed to line up the longitudinal axis with the x-axis, the forward end of the waterline at x = 0, and the aft end in the positive x direction.
