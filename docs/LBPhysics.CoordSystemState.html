Class: CoordSystemState

LBPhysics.CoordSystemState

new CoordSystemState() → {LBPhysics.CoordSystemState}

This is used to track the transforms for converting between a world coordinate system and a local coordinate system. It can also keep track of the previous transforms so it may calculate the world and local velocities of points in either coordinate system.

calcVectorLocalToWorld(localPos, results, prevLocalPos) → {LBPhysics.CoordSystemState}

Calculates world position, world velocity, and local velocity for a given local point.
Parameters:
Name Type Description
localPos object The local coordinates of the point of interest.
results object The object to receive the results. Results are stored in keys, and are only computed if the key exists: worldPos: The world coordinates of localPos. worldVel: The velocity of localPos in world coordinates. localVel: The velocity of localPos in local coordinates.
prevLocalPos object If defined, the previous local position, used for velocity calculation, if not defined then the previous position is presumed to be localPos.

setXfrms(worldXfrm, dt, localXfrm) → {LBPhysics.CoordSystemState}

Sets the world and local transforms, saving the previous transforms in the process. If either Xfrm matrix is not defined it is obtained by inverting the other matrix, if both are not defined then the matrices are set to the identity matrix.
Parameters:
Name Type Description
worldXfrm object If defined, the 4x4 matrix for transforming from local to world coordinaes.
dt number The simulation time change from the last call to this, used to compute velocity.
localXfrm object If defined, the 4x4 matrix for transforming from world to local coordinates.
