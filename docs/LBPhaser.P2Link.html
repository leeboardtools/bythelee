Class: P2Link

LBPhaser.P2Link

new P2Link(phaserEnv) → {LBPhaser.P2Link}

Manages linking a Phaser.Physics.P2.Body and a LBPhysics.RigidBody, updating the position/rotation of the rigid body from the P2 body and applying the forces from the rigid body to the P2 body. This also supports displaying an arrow representing the resultant force on rigid bodies.

updateForceArrowFromRigidBody(rigidBody, forceArrow) → {undefined}

Updates a force arrow from a rigid body's resultant. If the rigid body has a 'getForceArrowResultant' member function defined, that will be called to obtain the resultant, otherwise the rigid body's getResultant() method will be called. The getForceArrowResultant has the same function signature as LBPhysics.RigidBody#getResultant.
