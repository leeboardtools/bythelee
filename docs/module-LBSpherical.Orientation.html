Class: Orientation

LBSpherical.Orientation(azimuthDegopt, elevationDegopt, rotationDegopt) → {module:LBSpherical.Orientation}

new Orientation(azimuthDegopt, elevationDegopt, rotationDegopt) → {module:LBSpherical.Orientation}

Defines an orientation in spherical coordinates using azimuth, elevation, and rotation angles in degrees.

An azimuth of 0 degrees points towards the +x axis. An elevation of + degrees points towards the +z axis. The rotation angle is around the azimuth/elevation axis
