Class: Project3D

LBPhaser.Project3D

new Project3D(env, groupopt, cameraopt) → {LBPhaser.Project3D}

This is a hack projection of 3D objects onto the Phaser 2D surface using a Phaser.Graphics object.

This maintains a z-buffer of sorts, with panels defined in 3D space added for display. The panels can be filled and/or framed.

Parameters:
Name Type Attributes Description
env LBPhaser.PhaserEnv The Phaser environment
group Phaser.Group <optional>
 Optional group to which the projection display object will be attached.
camera LBCamera.Camer <optional>
 If defined the camera to use, otherwise a camera pointing down along the z-axis will be configured.

    addPanel(frameColoropt, lineWidthopt, closeFrameopt, fillColoropt, alphaopt, vertices, indicesopt, toWorldXfrmopt) → {LBPhaser.Project3D}

    Adds a panel for projection. Note that panels are presumed to be flat.

    Panel vertices should be ordered counter-clockwise when looking at the panel from the outside.

    Parameters:
    Name Type Attributes Default Description
    frameColor Number <optional>
    		 If defined the color for the frame, otherwise the frame is not drawn.
    lineWidth Number <optional>
    		 The width of the line for the frame, if undefined then the default for Phaser.Graphics#lineStyle will be used.
    closeFrame Boolean <optional>
    		 false If true and the frame is being drawn, this ensures that a final line from the last vertex to the first vertex is drawn, closing the frame.
    fillColor Number <optional>
    		 If defined the color to fill with, otherwise the panel is not filled.
    alpha Number <optional>
    		 1 If defined the alpha value for the frame and fill, otherwise the defaults for Phaser.Graphics will be used.
    vertices Array.<LBGeometry.Vector3> The array of vertices.
    indices Array.<Number> <optional>
    		 Optional array of the indices of the points in vertices to use for drawing, otherwise vertices is presumed to be in the desired order of the vertices.
    toWorldXfrm LBGeometry.Matrix4 <optional>
    		 If defined the transform used to convert the vertices to world coordiantes.
