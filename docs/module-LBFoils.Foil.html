Class: Foil

LBFoils.Foil() → {module:LBFoils.Foil}

new Foil() → {module:LBFoils.Foil}

A foil generates a 3D force in a fluid flow, using a 2D slice within a local coordinate frame to calculate the force. The 2D slice lies within the local x-y plane, while the foil's span lies along the z-axis.

The force generated by the foil is presumed to lie within the 2D slice's plane, therefore its local z coordinate will be the same as the z coordinate of the 2D slice in the local coordinate system.
