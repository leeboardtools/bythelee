(static) addTetrasToBody(body, tetras) → {CANNON.Body}

Adds tetras as shapes to a CANNON.Body. Once tetras have been added to a body, LBCannon.updateBodyCenterOfMass should be called to properly align the body's center of mass with the origin.
Parameters:
Name Type Description
body CANNON.Body The body to add the tetras to.
tetras Array.<LBVolume.Tetra> The array of tetras to be added.
