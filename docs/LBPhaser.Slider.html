Class: Slider

LBPhaser.Slider(game, lowerValue, lowerValuePixel, upperValue, upperValuePixel, isVertical, thumbSprite, bkgdObjectopt, lowerImageopt, lowerImageopt) → {LBPhaser.Slider}

new Slider(game, lowerValue, lowerValuePixel, upperValue, upperValuePixel, isVertical, thumbSprite, bkgdObjectopt, lowerImageopt, lowerImageopt) → {LBPhaser.Slider}

A Phaser based slider control, the control contains a thumb that moves along either the local x-axis (horizontal orientation) or the local y-axis (vertical orientation).

The control creates a group (LBPhaser.Slider#group) to which it adds the thumb and other drawing objects, the control can be positioned and rotated by directly accessing that group.

Parameters:
Name Type Attributes Description
game Phaser.game The Phaser game object.
lowerValue Number The value associated with one end of the controlled range.
lowerValuePixel Number The pixel value associated with lowerValue, this is the position of the sprite when the current value is lowerValue.
upperValue Number The value associated with the other end of the controlled range.
upperValuePixel Number The pixel value associated with upperValue, this is the position of the sprite when the current value is upperValue.
isVertical Boolean If true the thumb locally moves vertically, otherwise it moves horizontally.
thumbSprite Phaser.Sprite | Phaser.Image The sprite representing the thumb (the draggable part)
bkgdObject Object <optional>
 The background drawing object. If this is a its tile position will be updated to match the pixel position of the thumb.
lowerImage Object <optional>
 If defined, a label that appears outside the side of the control with corresponding to lowerValue.

 If defined, a label that appears outside the side of the control with corresponding to upperValue.
