LBControls.CSplineValueMapper

new CSplineValueMapper(splineData) → {LBControls.CSplineValueMapper}

A mapper object for use as the LBControls.SmoothController#offsetValueMapper that uses a simple cubic spline via LBMath.CSpline.

Name Type Description
