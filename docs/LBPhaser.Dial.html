Class: Dial

LBPhaser.Dial(game, lowerValue, lowerDeg, upperValue, upperDeg, thumbSprite, bkgdObjectopt) → {LBPhaser.Dial}

new Dial(game, lowerValue, lowerDeg, upperValue, upperDeg, thumbSprite, bkgdObjectopt) → {LBPhaser.Dial}

A Phaser based rotary dial control. The control contains a thumb that rotates about the local origin.

The control creates a group (LBPhaser.Dial#group) to which it adds the thumb and other drawing objects, the control can be positioned and rotated by directly accessing that group.
