LBControls.SteppedController

new SteppedController(nameopt, stepsopt, initialValueopt, controlleeopt) → {LBControls.SteppedController}

A controller that takes on a discrete set of values. The actual value of the controller is an index into the discrete set of values.

Members

controllee

The function that is called whenever the current value of the controller is changed. The function should have the signature:

controllee = function(value, controllee) {}

where value is the object of the steps array (you can obtain the index from controllee.getValue()).
