(static) loadCylinderFromData(data, massopt) → {LBVolume.Cuboid|undefined}

Loads a cylindrical volume from properties in a data object. For now it simply loads it as a LBVolume.Cuboid.
Parameters:
Name Type Attributes Description
data Object The data object. This should contain the following properties:
  • base: {"x":x, "y":y, "z":z} The base of the cylinder, it will be extruded in the +z direction.
  • size: {"x":dx, "y": dy, "z": length} The dimensions of the cylinder. dx is the diameter along the x-axis, dy is the diameter along the y-axis, length is the length of the cylinder, which is along the +z axis.
  • [pos]: See LBGeometry.loadVector3
  • [rotation]: See LBGeometry.loadEuler If specified, the rotation to apply to the vertices.
    		•
    mass Number <optional>
    		 The mass of the cylinder.
