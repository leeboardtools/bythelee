Class: ChaseCameraController

LBUI3d.ChaseCameraController(distanceopt, chaseModeopt, globalLimitsopt) → {module:LBUI3d.LocalPOVCameraController}

new ChaseCameraController(distanceopt, chaseModeopt, globalLimitsopt) → {module:LBUI3d.LocalPOVCameraController}

A camera controller that acts as a chase, it follows the target at a given location relative to the target's local coordinate system. This differs from module:LBUI3d.LocalPOVCameraController in that for this the camera is looking towards the target, whereas for the first person controller the camera is looking from the target.

How the controller updates the camera's position and orientation is determined by the chase mode.

For module:LBUI3d.ChaseCameraController.CHASE_MODE_LOCAL, the controller tries to update the camera such that the target stays fixed relative to the camera. The location of the camera is specified relative to the target via module:LBUI3d.ChaseCameraController#desiredCoordinates, which is a module:LBSpherical.CoordinatesRAA. The reference orientation of the camera is based upon a local reference orientation frame relative to the target's local coordinates where the x axis points to the target and the z axis is in the same direction as the target's z axis when the elevation and azimuth angles are zero. module:LBUI3d.ChaseCameraController#desiredOrientation is then added to this local reference orientation to obtain the camera's orientation in the target's local coordinate system.

For module:LBUI3d.ChaseCameraController.CHASE_MODE_WORLD, the controller tries to update the camera such that the camera maintains a fixed world height, but in the horizontal plane stays at the same azimuth angle relative to the target's local coordinates. The reference orientation of the camera is based upon a reference orientation frame that is pointed at the target. This reference frame is in world coordinates. module:LBUI3d.ChaseCameraController#desiredOrientation is then added to this orientation reference frame to obtain the camera's orientation in world coordinates.

Rotation/zoom have the effect of moving the location of the camera relative to the target's local coordinate system, while panning has the effect of changing the orientation of the camera within its local orientation frame.
