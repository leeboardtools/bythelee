(static) addVolumesToBody(body, volumes, volToBodyXfrmopt) → {CANNON.Body}

Adds volumes as shapes to a CANNON.Body. Once volumes have been added to a body, LBCannon.updateBodyCenterOfMass should be called to properly align the body's center of mass with the origin.
Parameters:
Name Type Attributes Description
body CANNON.Body The body to add the tetras to.
volumes Array.<LBVolume.Volume> The array of tetras to be added.
volToBodyXfrm LBGeometry.Matrix4 <optional>
 If defined the transform to apply to each vertex to bring it to the coordinate system of the body.

Updates the center of mass of a body. The local origin of the body is the center of mass.

 The mass, if undefined the mass will not be modified.
