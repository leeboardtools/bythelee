Class: RollingBuffer

LBUtil.RollingBuffer(maxSize) → {module:LBUtil.RollingBuffer}

new RollingBuffer(maxSize) → {module:LBUtil.RollingBuffer}

A rolling indexable buffer with a maximum size, the oldest element is always at index 0, the newest at index module:LBUtil.RollingBuffer#getCurrentSize. Elements are added to the end of the buffer via module:LBUtil.RollingBuffer#push, removed from the end of the buffer via module:LBUtil.RollingBuffer#pop_newest, and removed from the front of the buffer via module:LBUtil.RollingBuffer#pop_oldest.
