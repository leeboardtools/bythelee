Namespace: LBGeometry

LBGeometry

The geometry components are primarily based upon the THREE.js geometry objects. The main purpose of our set of components is to isolate the direct references to THREE.js, the rest of Leeboard should not make direct references to THREE.js. This way should we want to something other than the full blown THREE.js we can simply replace this file.
