Class: View3D

LBUI3d.View3D(scene3D, container, cameraopt, rendereropt) → {module:LBUI3d.View3D}

new View3D(scene3D, container, cameraopt, rendereropt) → {module:LBUI3d.View3D}

A view on a scene. The view provides the association between a DOM element to display the view of a scene into, and a camera to view through.

Views also provide support for one or more module:LBUI3d.CameraControllers, which are used to control the camera within the view.
